🌦️ Weather Dashboard – ReactJS (Open-Meteo API)
A responsive Weather Dashboard built using ReactJS that displays current weather, hourly forecasts, air quality data, and historical weather trends using the Open-Meteo API. The application automatically detects the user’s location using the browser Geolocation API and shows localized weather data with interactive charts.

🚀 Features

Automatic GPS location detection
Current weather information
Hourly weather charts
Air quality data (PM10, PM2.5, CO, NO2, SO2)
Historical weather data (date range up to 2 years)
Interactive charts with zoom and scroll
Celsius ↔ Fahrenheit temperature toggle
Fully responsive dashboard UI
Fast API data loading


📊 Weather Data Includes

Current, Min & Max Temperature
Sunrise & Sunset
Wind Speed
Relative Humidity
Precipitation
UV Index
Visibility
Air Quality Data

Historical charts include temperature trends, precipitation, wind data, sunrise/sunset, and air quality trends.
